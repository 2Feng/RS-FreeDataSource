免费遥感数据源介绍¶

前言¶

欢迎来到这里！免费的遥感数据源现在有多种，但能查阅的中文介绍资料不多，关于下载和使用方法的教程也比较零散。为此我整理了本文档，目的在于帮助大家了解和随时查阅各个免费数据源的情况，以便更好地学习利用这些数据。

注解

由于资料有限，本文档初期只能覆盖常用的几个数据源，欢迎点击右上角 Edit on GitHub ，帮助完善或更新本文档内容。

数描述范围¶

主要介绍各种数据的以下几个方面：

  • 数据概况
  • 波段介绍
  • 下载地址
  • 数据样例
  • 常用领域
  • 使用案例（可选）
